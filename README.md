# CodeAlpha Machine Learning Internship Projects

## Overview

This repository contains the Machine Learning projects completed as part of the **CodeAlpha Machine Learning Internship Program**. Each project focuses on solving real-world problems using Machine Learning techniques and demonstrates the complete workflow from data preprocessing to model evaluation and prediction.

---

# 📂 Projects Included

## Task 1: Credit Scoring Model

### Description

A Machine Learning model that predicts whether a loan application is likely to be approved based on applicant information such as income, education, credit history, loan amount, and property area.

### Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Joblib
- Google Colab

### Features

- Data Preprocessing
- Missing Value Handling
- Feature Encoding
- Data Visualization
- Logistic Regression Model
- Model Evaluation
- Loan Approval Prediction
- Model Saving
