
# Heart Disease Prediction

## 📌 Project Overview

This project predicts whether a patient is likely to have heart disease using Machine Learning. It was developed as part of the CodeAlpha Machine Learning Internship.

---

## 🎯 Objective

Build a classification model that predicts the presence of heart disease based on patient medical information.

---

## 📂 Dataset

- Dataset: UCI Heart Disease Dataset
- Total Records: 920
- Features Used: 13
- Target:
  - 0 → No Heart Disease
  - 1 → Heart Disease

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib

---

## 🤖 Machine Learning Algorithm

- Logistic Regression

---

## 📊 Project Workflow

1. Load Dataset
2. Handle Missing Values
3. Encode Categorical Features
4. Feature Scaling using StandardScaler
5. Train-Test Split (80:20)
6. Train Logistic Regression Model
7. Evaluate Model
8. Save Trained Model
